# NASA Picture of the Day 🚀

This project displays NASA's Astronomy Picture of the Day using the NASA APOD API.

Users can:
- View today's NASA image automatically
- Search for images by date
- Save previous searches in local storage
- Click history items to view images again

## Technologies Used
- HTML
- CSS
- JavaScript
- NASA APOD API
- LocalStorage
